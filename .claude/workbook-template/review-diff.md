## Patterns Summary

1. **Column group shift left by 1 (Fixas, Variáveis, Adicionais, Extras):** Per-month data group moved from starting at col D to col C (Item/Data/Valor = C/D/E instead of D/E/F). All numfmts, formulas, and month-banner merges updated accordingly. Last column (AM) eliminated.

2. **Freeze pane correction (Fixas, Variáveis, Adicionais, Extras):** `C2`/`D2` → `C3` — freeze moved down one row to lock both the month-banner row and the column-header row simultaneously.

3. **Mês banner extended to span 2 rows (all sheets with month header):** `A1:B1` merge → `A1:B2` — the Mês label now vertically spans row 1 (month names) and row 2 (Item/Data/Valor headers), consistent with the freeze pane covering row 2.

4. **Category and subcategory labels merged vertically (all expense sheets):** Instead of repeating `A='Habitação', B='Diarista'` in every data row, the label is now placed once in the first row of the group and merged downward. Applied to both Category (col A) and Subcategory (col B) label columns.

5. **Placeholder category-label rows eliminated (Receitas, Variáveis):** Rows that contained only a repeated category label (to visually fill the group) were removed. Replaced by vertical merges. Row count reduced by 2–3 rows per sheet.

6. **SUM formulas shifted left by 1 to match new Valor column (Fixas, Variáveis, Adicionais, Extras):** Every `=SUM(F3:F5)` style formula updated to `=SUM(E3:E5)` (and equivalently across all 12 months) to reference the new Valor column position.

7. **Receitas: separator row removed and block restructured:** The blank black-filled separator row between Salário and 13° blocks (row 8) removed. 13° block renumbered. Category-label rows eliminated. Salário subcategory merge extended to cover total row.

8. **Listas de itens: label column count reduced from 5 to 3 (A/B/C):** Original had 5 label cols (sheet-type, category, subcategory, item, blank) with data starting at F. Reviewed uses 3 cols (sheet-type, subcategory, item) with data starting at D. Freeze moved from F4 to D4. Month columns shifted from F–Q to D–O.

9. **Listas de itens: formula values populated in all data cells:** All month-value cells that were blank in the original now contain cross-sheet reference formulas (referencing corresponding cells in Fixas, Variáveis, etc.). Applies to ~176 cells across all expense rows.

10. **Listas de itens: sheet-type section labels added (col A, rows 28–58):** Col A now carries the sheet name ('Variáveis', 'Extras', 'Adicionais') as a merged vertical label for each expense section, replacing blank cells. Complements existing 'Fixas' label.

11. **Number format corrections on Listas summary rows:** Investimentos total and Salário/13° totals gain `R$ #,##0.00`; `% sobre Receita` rows gain `0.00%`. ~228 numfmt changes across Listas.

12. **Receitas total-row fill changed:** Row with `FFF2F2F2` (light pink) fill retained for total rows; the separator row's `FF000000` (black) fill cleared entirely when row was removed.

13. **Sub-item label column (col C) removed; subcategory + sub-item consolidated into col B (Variáveis):** Original used col A = category, col B = subcategory, col C = sub-item label (e.g. `'Orion'` + `'Consultas'`). Reviewed drops col C and moves the combined label into col B as `'Orion - Consultas'` (dash-separated). Col C is now the first data column (Item), completing the shift-left by 1.

14. **Listas % sobre Receita row fill:** Changed from `FFC0C0C0` (silver, same as header) to `FFCCCCFF` (light purple) — visually differentiates the percentage row from header rows.
