## Design Details (consolidated from QUICK.md 2026-07-01)
- **Cell inclusion:** a cell is emitted if it has a value, a formula, OR a non-default
  style (keeps styled-but-empty cells). Iterates `GetRows` output → captured only up to
  the last valued cell per row (trailing empties dropped).
- **`rowType` classifier:** text/formula signals first (month names, Item/Valor, "Total",
  SUM), palette as corroboration — degrades on off-palette sheets. Types: header-month,
  header-col, total-row, data-row, category-label, separator.
- **`rowFill` / separator rows:** otherwise-empty rows carrying a ROW-LEVEL fill (black
  separator bands) are resurrected by probing an empty cell's style (`GetCellStyle`
  falls back to the row style). Added commit `906a99d`.
- **crossSheetRefs:** parsed from FORMULAS (not values), handling `'Quoted Name'!Ref`.
  ODS dot-notation strings (`'Listas de itens'.F37` inside CONCATENATE) are NOT counted —
  constructed strings, not real refs.
